# Sistema de Gestión de Reencauche de Llantas

## Descripción del emprendimiento
Este sistema está diseñado para talleres y negocios dedicados al reencauche de llantas, con el objetivo de facilitar la gestión de clientes y el control de inventario de llantas. Permite optimizar los procesos de almacenamiento, búsqueda y registro de llantas de manera rápida y sencilla.

## Problema que resuelve el sistema
Antes de implementar este sistema, la organización y agregación de nuevos clientes era lenta y manual, y la búsqueda y control de llantas resultaba complicada. Con esta herramienta, se pueden registrar clientes, almacenar información de llantas y buscar productos específicos dentro de la misma interfaz de manera eficiente.

## Tecnologías utilizadas
- PHP  
- SQLite  
- HTML y CSS  

## Estructura del proyecto

/proyecto-reencauche-llantas
│
├─ /assets # Imágenes y capturas de pantalla
├─ /src # Código PHP principal
├─ /database # Base de datos SQLite
│ └─ database.sqlite
├─ README.md # Documentación del proyecto
└─ index.php # Página principal del sistema


## Breve explicación del código
- **index.php:** Interfaz principal para registrar clientes y buscar llantas.  
- **src/**: Funciones y clases PHP que manejan el registro, actualización y búsqueda de datos.  
- **database/database.sqlite:** Base de datos que guarda la información de clientes y llantas.  
- **assets/**: Carpeta donde se almacenan imágenes y capturas del proyecto.
